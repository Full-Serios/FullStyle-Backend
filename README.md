# Full-Style Backend

### Instrucciones
Se crea el entorno virtual
```bash
py -3 -m venv .venv
```
Se activa el entorno virtual
```bash
.venv\Scripts\activate
```
Se instalan las dependencias
```bash
pip install -r requirements.txt
```
Se inicia la aplicaci√≥n
```bash
python app.py
```
La app se desplegara en: `127.0.0.1:5000`
